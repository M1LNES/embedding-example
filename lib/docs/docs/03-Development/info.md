## File Extension

| Extension | Explanation                                                     |
| --------- | --------------------------------------------------------------- |
| \*.jsx    | React components                                                |
| \*.js     | Files with constants/functions for front-end or back-end routes |
| \*test.js | Unit tests                                                      |

## Case Types

| Type               | Case Type  |
| ------------------ | ---------- |
| Back-end endpoints | kebab-case |
| File names         | kebab-case |
| Variable names     | camelCase  |
